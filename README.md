# النظام الخبير للفقه المقارن
## مشروع ماستر الشريعة الإسلامية — تقنية الذكاء الاصطناعي

---

## 📋 نظرة عامة

نظام خبير ذكي يُقارن آراء المذاهب الفقهية الأربعة في المسائل الشرعية، مدعوماً بتقنيات الذكاء الاصطناعي ومعالجة اللغة الطبيعية العربية.

---

## 🗂️ هيكل الملفات

| الملف | الوظيفة |
|---|---|
| `01_database_schema.py` | هيكل قاعدة البيانات الفقهية (MongoDB Schema) |
| `02_db_operations.py`   | عمليات البحث والاسترجاع والإحصاء |
| `03_inference_engine.py`| محرك الاستنتاج — معالجة النص + البحث |
| `04_rules_engine.py`    | نظام القواعد الأصولية الكبرى |
| `05_rag_system.py`      | نظام RAG المتكامل (Retrieval + Generation) |
| `06_api_integration.py` | دمج Claude API / GPT-4 / Offline |
| `07_fiqh_ui.html`       | واجهة المستخدم الكاملة |
| `main.py`               | نقطة الدخول الرئيسية |

## 🚀 تشغيل النظام

### 1. تشغيل سريع (سطر الأوامر)
```bash
python main.py
```

### 2. واجهة المستخدم
افتح الملف `07_fiqh_ui.html` في أي متصفح — يعمل بدون خادم.

### 3. استخدام برمجي
```python
from main import FiqhExpertSystem

system = FiqhExpertSystem()
result = system.ask("ما حكم الجمع بين الصلاتين في السفر؟")
system.display(result)
```

### 4. مع Claude API
```python
from api_integration import ClaudeFiqhGenerator
from rag_system import FiqhRAGSystem

rag = FiqhRAGSystem(MASAIL_DB, api_key="sk-ant-...")
result = rag.ask("ما حكم الصلاة خلف الإمام الفاسق؟")
```

## 🔭 التطوير المستقبلي

1. **توسيع قاعدة البيانات** إلى 500+ مسألة من كتب الفقه المقارن
2. **AraBERT** لتحسين دقة البحث الدلالي إلى >90%
3. **ChromaDB أو FAISS** لقاعدة المتجهات الكاملة
4. **واجهة ويب** بـ Flask + React
5. **تكامل مع مكتبة شاملة** كالموسوعة الفقهية الكويتية
